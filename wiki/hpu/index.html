HARD PLATFORMER ULTIMATE is a modified version of HARD PLATFORMER, this mod is based on version 1.1.0 of HP and made by RedShadow, RedShadow is Azapru's (original HP dev) friend.

This modification changes and fixes a lot of thinks about original game.









Changes made in this mod

- 2 Player races: basically local multiplayer, in this mode "second player" color is red instead of green like in online multiplayer.

- Better menus: This mod completely remakes the menus used by the original game adding backgrounds and more features. (NOTE FROM AZAPRU: "yeee original menus were trash, that's because i didn't know this game gonna evolve to series, i originaly made this as some dum smoll gaem")

- Online fixes

- Full controller support: The use of controllers is remade in this mod. Controller support in original game was unfinished.

- Saving fixes: This modification fixes some issues with progress saving.

- Keybinds

- Auto updater: while all games made by Azapru after HP have auto-updater/update checker, Original HP doesn't have one, HPU adds that.
