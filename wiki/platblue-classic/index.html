HARD PLATFORMER (also known as Platblue Classic) is a difficult platformer game released in January 2021 as full version on GameJolt and later and Itch.io for Windows. Everything in this game (sprites, sfx, code) was made by only one person, Azapru. First versions released on GameJolt were unfinished alpha versions, there were 5 alpha releases, on January 20, the first full game version was released (v1.0.0).

In this game you are completing rooms (levels) with obstacles and other items, the game has 25 levels (+4 special levels, and 1 unused).









Gameplay

In HP player use arrow keys to move and Z key to jump, player starts in entrence on left side of the room and need to complete level through exit on right side of the the room.

After completing 5 levels, new rooms gonna have different color. First room is green and last playable room is dark grey. Each new room color gets harder and have some kind of new item.












Challenges

Challenges are special levels with some special gimmick like time limit or jump limit.

There are only 3 challenges: BEAT LEVEL IN 5SEC, BEAT LEVEL WITHOUT TOUCHING SLIME and NO DOUBLE JUMP.













Multiplayer

In HARD PLATFORMER you can join a multiplayer level where you can find other players, you can join own lobby or public lobby. Originally multiplayer level was really small but in later update it got enlarged.

This mode has special item called "spawn point light", if you touch it, it will activate, once you die you will respawn at last activated spawn point light.

Multiplayer have anti-cheat that can detect and kick if player is flying (using cheat engine for example) or is idle for too long. Anti-cheat is on softwere level meaning it can be disabled or modified using source code and also cannot ban players.









Level Editor

In level editor you can create your own levels, it's very limited and lacks a lot of features and items

Editor have only 4 items: block, vertical laser-chain, horizontal laser-chain and slime block

Players had option to share levels through online level browser, it's hosted on https://zetifox03.wixsite.com/hplevelbrowser, while online level browser is still up and you can download levels from there, you cannot sumbit your levels anymore.











Bugs

- Font is broken on most devices, this is clickteam fusion 2.5 bug

- Level editor causes crashes and black screens on some devices

- Changing server causes game to break until reloaded















End of support & source code publication

On June 2, 2021, Azapru posted that she starts working on HARD PLATFORMER PLUS (now known as Platblue)

On November 25, 2021, Azapru posted an article saying that HARD PLATFORMER PLUS will be changing its name and all HP websites will be shut down (but website shutdown never happened).

On March 1, 2022, Azapru posted another article about HP, she said that HP is not supported anymore and will not reveive any more updates. In this post she also released source code. Some time later post was edited to inform that HP remake (Platblue/HP PLUS) is now work in progress.









HARD PLATFORMER Blue

HARD PLATFORMER Blue was going to be a spin-off game with less content then original game, it had 5 exclusive levels with blue room color. Original build is now lost.

All HPBlue exclusive levels were added to main game in update 1.0.0 (after alpha) as bonus levels.
















Android Port

Android port was never released because of engine limitations and non-fixable bugs. In this port everything was blurred, colission were incorrect or completly broken, multiplayer was causing crashes and it was impossible to log-in.

Original .apk is now lost, fun fact: it had guest login!
